#UI

##这个UI是什么?
剧中人在个人博客以及项目中反复使用的功能的公用视觉类的代码集合，

不是一个大而全的东西，仅仅是根据自己的需求整理，如果你对我的代码感兴趣，也欢迎你来使用！

##UI目前有哪儿些内容？

* 目前有虚拟弹层、弹框、提示框、确认框等功能
* 支持对象化事件的调用
* 支持窗体自由`拖拽`
* 支持`自定义位置`，方便控制对象在页面中的呈现
* 对象被注销有`回调支持`，方便确认对象
